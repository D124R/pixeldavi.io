# TypeScript Calculator

A simple calculator project with an HTML frontend and Node.js backend built in TypeScript.

## Features

- Browser-based calculator UI
- REST API for expression evaluation
- TypeScript-powered backend and frontend build

## Getting Started

1. Install dependencies

```bash
cd calculator-app
npm install
```

2. Run in development mode

```bash
npm run dev
```

3. Build and start for production

```bash
npm run start
```

4. Open your browser at `http://localhost:3000`
